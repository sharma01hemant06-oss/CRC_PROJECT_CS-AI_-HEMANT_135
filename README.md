# 🎓 Student Grade Management System

## 📌 Description
This is a Java-based Student Grade Management System that stores student details such as name, roll number, and marks.  
It calculates the GPA using loops and assigns grades based on performance.

## 🛠️ Features
- Store student name and roll number  
- Accept multiple subject marks  
- Calculate average marks  
- Convert percentage to GPA  
- Display grade (A, B, C, D, F)

## 💻 Technologies Used
- Java  
- OOP Concepts  
- Loops and Arrays
